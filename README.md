# Business Card App

A modern, stylish Android Business Card application built using Kotlin and Jetpack Compose.

## Features

- **Material 3 Design**: Utilizing modern UI components and color schemes.
- **Card-based Layout**: Elegant presentation using elevated cards.
- **Gradient Background**: A dynamic visual experience with vertical gradients.
- **Vector Icons**: Integrated Material design icons for contact information.
- **Responsive Design**: Clean and centered UI for various screen sizes.

## Built With

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

## Project Structure

- `MainActivity.kt` → App entry point and UI logic.
- `BusinessCard()` → Main UI composable defining the card and layout.
- `ContactInfoRow()` → Reusable component for displaying contact details with icons.
